# opaScroll
OnePAgeScroll

A Simple Boilerplate for beautiful One Page Scrolling 

## Parameter
```javascript
//set time to wait after scroll
var time = 200;

//set the height to scroll
var height = 0;

//ids to jump to
var ids = [];

//set animation duration
animationTime = 600;

//sroll to top at load
scrollToTop = false;
```
